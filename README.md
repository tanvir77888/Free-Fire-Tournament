# Tanvir Shop - Gaming Top-up Web App

### 🛠 প্রযুক্তি (Tech Stack):
- **Frontend**: HTML5, Tailwind CSS, JavaScript (Vanilla)
- **Backend**: PHP 8.x
- **Database**: MySQL
- **Icons**: Font Awesome
